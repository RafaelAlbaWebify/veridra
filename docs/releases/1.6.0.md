# Veridra 1.6.0

- Adds explicit loopback-local client projects.
- Binds a project name, client label, public website, and optional report profile.
- Provides one-click assessment, report, evidence export, and history-save actions.
- Supports deterministic atomic create, edit, list, load, and delete workflows.
- Keeps default Veridra reporting when a project has no assigned profile.
