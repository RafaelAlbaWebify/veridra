# Veridra 2.5.0

## Validated crawl profiles

- Adds Quick, Standard and Deep named crawl profiles.
- Adds a guarded Custom profile with server-side hard ceilings.
- Keeps anonymous/free assessment behavior fixed to the conservative Quick profile.
- Records the selected profile and every effective crawl limit as assessment evidence.

## Operator delivery routes

- Adds profile-aware assessment, HTML report, PDF report and evidence ZIP routes under `/crawl`.
- Preserves saved report-profile branding for HTML, PDF and ZIP delivery.
- Rejects unknown profiles and out-of-range custom values before website collection begins.

## Saved projects

- Saves the selected crawl profile with each client project.
- Supports persisted custom page and depth values.
- Adds project assessment, report and export routes that always use the project’s stored crawl profile.
- Crawl configuration participates in deterministic project identity.

## Resource controls

Crawling remains sequential. Hard limits apply to page count, depth, total bytes, per-page bytes, timeout, sitemap files and sitemap URLs. This release does not add scheduling, concurrency, background workers, accounts or distributed quotas.
