# Veridra 3.2.0

Veridra 3.2.0 completes the first broad entitlement integration over the local workspace policy foundation.

## Added

- Project-count enforcement before new project creation.
- White-label entitlement checks for report-profile creation, editing and branded report use.
- Embedded lead-form entitlement checks for form creation and public audit submission.
- Lead-submission quota reservation and successful-use recording.
- Monitoring-run quota reservation and successful-use recording.
- Legacy PDF and evidence-export quota enforcement and metering.
- Webhook and email retry-attempt metering.
- Controlled 403 and 429 responses from the composed runtime middleware.
- Explicit isolation for anonymous `/free/*` routes and already-metered `/crawl/*` routes.

## Compatibility

The middleware remains inactive until a workspace plan is explicitly saved. Existing local installations therefore retain their previous behavior until the operator activates plan enforcement.

## Boundaries

This release is a local policy and evidence layer. It does not provide authentication, multi-tenant isolation, payment collection, Stripe integration, tax handling or production subscription security.
