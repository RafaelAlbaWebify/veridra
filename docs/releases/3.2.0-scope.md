# Veridra 3.2.0 scope

This release integrates the 3.1.0 workspace policy across commercial routes through one composed-runtime middleware.

Covered operations:

- project creation;
- report-profile creation and editing;
- branded report/profile use;
- embedded audit-form creation and submission;
- lead-submission usage;
- project monitoring runs;
- legacy PDF generation;
- legacy evidence export;
- explicit webhook and email retries.

Excluded from this release:

- authentication;
- tenant isolation;
- payment-provider integration;
- taxes and invoicing;
- automatic background processing;
- public SaaS deployment claims.
