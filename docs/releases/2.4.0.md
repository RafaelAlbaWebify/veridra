# Veridra 2.4.0

Veridra 2.4.0 adds bounded server-generated PDF reports using the existing escaped report HTML and a headless Chromium renderer.

## Added

- `/report.pdf` download route with the same target and report-profile selection as HTML reports.
- A4 PDF rendering with print backgrounds, bounded input/output sizes, a 20-second render timeout, repeated footer and page numbers.
- Safe deterministic filenames, `application/pdf`, `nosniff` and `no-store` response headers.
- Remote-request blocking during PDF rendering; reports do not fetch external logos, fonts, stylesheets or arbitrary assets.
- Structural route tests and a real Chromium PDF smoke test in CI.

## Scope and limitations

The Chromium browser binary must be installed separately with Playwright's browser installer. Rendering is bounded and fails cleanly when Chromium is unavailable. This release does not claim PDF/A conformance, tagged-PDF accessibility, editable PDF content, or pixel-identical output across browser/platform versions. Existing HTML reports and evidence ZIP packages are unchanged.
