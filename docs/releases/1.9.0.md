# Veridra 1.9.0

- Adds operator-triggered reassessment for saved client projects.
- Saves each distinct project run into local assessment history.
- Shows project-specific latest and previous runs.
- Automatically compares added, resolved, changed, and unchanged findings.
- Provides project timelines and links to full saved assessments and comparisons.
- Keeps monitoring local and manual; no scheduler, accounts, cloud storage, or email delivery.
