# Veridra 2.3.0

## Multi-page quality findings

This release adds four deterministic checks over already collected crawl evidence:

- duplicate non-empty document titles;
- duplicate non-empty meta descriptions;
- image elements missing an `alt` attribute;
- multi-hop redirect chains;
- oversized decoded HTML bodies above a 500,000-byte review threshold.

Duplicate groups include the repeated value, affected URLs and count. Missing metadata remains handled by the existing missing-title and missing-description findings and is not treated as duplicate content.

The image check distinguishes a missing `alt` attribute from an explicit empty `alt`, which may be appropriate for decorative images.

Redirect evidence includes requested URL, final URL, bounded redirect chain and hop count.

Page-size evidence is explicitly described as the decoded HTML body encoded as UTF-8. It is not represented as browser transfer size, compressed size, rendered weight or performance scoring.

All checks use the existing bounded, sequential, same-origin crawl. No JavaScript execution, authentication, form submission, active scanning or external data provider is introduced.
