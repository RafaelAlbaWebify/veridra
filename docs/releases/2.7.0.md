# Veridra 2.7.0

Veridra 2.7.0 adds explicit local monitoring schedules and bounded due-run operations for saved client projects.

## Added

- Manual, daily, weekly and monthly monitoring schedules.
- IANA timezone, preferred local run time, weekday and day-of-month settings.
- Deterministic timezone-aware next-due calculations.
- Monitoring operations page showing manual, upcoming, due and overdue projects.
- Individual project run action.
- Bounded sequential due-project batch execution with a hard 20-project limit.
- Per-project failure isolation and successful assessment persistence in existing history.
- Schedule editing with atomic project replacement.
- Preservation of each project's crawl and report profiles.

## Boundaries

Schedules are evaluated only while Veridra is running and a user triggers the operation. This release does not install a daemon, cron task or cloud scheduler and does not claim unattended monitoring, email alerts or automatic report delivery.
