# Veridra 3.2.0 validation checklist

- Workspace policy remains inactive until explicitly configured.
- Free tools bypass local paid-workspace counters.
- Profile-aware crawl routes avoid double metering.
- Commercial feature denials return controlled 403 responses.
- Quota exhaustion returns controlled 429 responses.
- Successful metered routes append usage events.
- Failed routes do not append successful-use events.
- Exact-head Ruff, strict mypy, pytest, deterministic audit and Chromium browser audit must pass before merge.
