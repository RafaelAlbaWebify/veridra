# Veridra 3.1.0

Veridra 3.1.0 adds a local workspace-plan and usage-policy foundation.

## Added

- Explicit Free, Solo, Professional and Agency plan catalogue.
- Local workspace configuration with plan, status and billing-cycle anchor.
- Append-only usage ledger for audits, crawl-page reservations, PDFs, exports, lead submissions, monitoring runs and delivery attempts.
- Deterministic quota decisions with used, requested, remaining and rejection reason.
- Operator workspace dashboard, plan preview, explicit local plan update and usage CSV export.
- Explicit policy activation: existing installations remain in compatibility mode until a workspace plan is saved.
- Crawl assessment, PDF and evidence-export quota enforcement after activation.
- Conservative crawl-page accounting based on reserved profile capacity.

## Commercial boundary

This release models and enforces local policy only. It does not provide authentication, tenant isolation, Stripe billing, payment collection, tax handling, invoices or production subscription enforcement.

## Metering boundary

Crawl-page usage is recorded as the maximum page capacity reserved by the selected crawl profile, not as browser-transfer billing or an estimate of infrastructure cost. Failed audits are not recorded as completed audit usage.
